Introducción

El módulo de importación Excel permite cargar datos masivamente al sistema desde archivos Excel (.xlsx, .xls).

Este manual te guiará paso a paso en el proceso de importación.

Requisitos

Formato del Archivo
  • Archivo Excel (.xlsx, .xls)
  • Tamaño máximo: 5MB
  • Sin macros ni contenido ejecutable

Proceso de Importación

  1. Haz clic en "Seleccionar archivo" o arrastra el archivo a la zona indicada
  2. Selecciona el tipo de importación
  3. Verifica la previsualización de los datos

  • El sistema validará automáticamente los datos
  • Se mostrarán advertencias y errores si los hay
  • Corrige los errores señalados antes de continuar

  • Haz clic en "Procesar"
  • Espera a que finalice el procesamiento
  • Revisa el resumen de la importación

Validación de Datos

Campo Validación Ejemplo
DNI 8 dígitos numéricos 12345678
Email Formato válido de email usuario@dominio.com
Fecha Formato DD/MM/YYYY 25/12/2024

Manejo de Errores

Errores Comunes
Error de formato
El archivo no cumple con el formato requerido
Datos inválidos
Los datos no cumplen con las validaciones
Archivo demasiado grande
El archivo excede el tamaño máximo permitido
